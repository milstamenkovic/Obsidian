Mil Obsidian OS - Custom Obsidian theme as a pseudo-OS
______________________________________________________



HOW TO USE "Mil Obsidian OS":

1) There are TWO files.
    - First file is DESKTOP. This is a .md file that is a note, which will represent OS desktop.
    - Second file is SNIPPET CSS. This is a .css file that is a CSS snippet that will make OS look like OS.
2) Put DEKSTOP in VAULT, anywhere. Put SNIPPET CSS in .obsidian snippet folder.
3) Set up Obsidian to always open to DESKTOP note.
4) Customize further on your own.
